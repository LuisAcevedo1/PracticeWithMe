JavaScript Let

La palabra clave let se introdujo en ES6 (2015)

Las variables declaradas con alcance let de bloque

Las variables declaradas con let deben declararse antes de su uso.

Las variables declaradas con let no se pueden volver a declarar en el mismo ámbito

Alcance del bloque

Antes de ES6 (2015), JavaScript no tenía Block Scope.

JavaScript tenía alcance global y alcance de función.
