Formas de mostrar en pantalla salidas

EN javaScript en pantalla tenemos diferentes formas de mostrar salidas

Escribiendo dentro de los elementos HTML usando innerHTML

Escribiendo como salida en html usando documunt.write()

Escribiendo una alerta de texto usando window.aler()

Escribiendo dentro de la consola del navegador usando console.log()

Usando innerHTML

para usar innerHTML se tiene que llamar desde un archivo javaScript al elemento donde queramos mostrar el elemento en este caso por ejemplo con un ID

Usando document.write()

Document.write() es otra forma de escribir dentro del DOM de html, para ello se debe de saber que el elemento a mostrar se vera en donde se llame a la funcion y no en otro lugar o si no es una funcion se llamara al final del archivo.
