Margenes CSS

Las propiedades margin CSS se utilizan para crear espacio alrededor de los elementos, fuera de los bordes definidos.

Con CSS, tienes control total sobre los márgenes. Hay propiedades para establecer el margen para cada lado de un elemento (superior, derecho, inferior e izquierdo).

Margenes Individuales en CSS

CSS tiene propiedades para especificar el margen para cada lado de un elemento:

margirn-top

margirn-right

margirn-bottom

margirn-left

Todas las propiedades de margen pueden tener los siguientes valores:

auto - el navegador calcula el margen

longitud : especifica un margen en px, pt, cm, etc.

% : especifica un margen en % del ancho del elemento contenedor

heredar: especifica que el margen debe heredarse del elemento principal

Margen Shorthand

Para acortar el código, es posible especificar todas las propiedades de margen en una sola propiedad.

El valor automatico

Puede configurar la propiedad de margen para autocentrar horizontalmente el elemento dentro de su contenedor.

Luego, el elemento ocupará el ancho especificado y el espacio restante se dividirá en partes iguales entre los márgenes izquierdo y derecho.

Colapso del margen CSS

A veces dos margenes colapsan en uno solo

Los márgenes superior e inferior de los elementos a veces se contraen en un solo margen que es igual al mayor de los dos márgenes.

¡Esto no sucede en los márgenes izquierdo y derecho! ¡Solo márgenes superior e inferior!
