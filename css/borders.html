Bordes

Las propiedades del borde CSS le permiten especificar el estilo, ancho y color del borde de un elemento.

Estilo de bordes en CSS

dotted - Define un borde de puntos

dashed - Define un borde descontinuo

solid - Define un borde solido

double - DEfine un borde doble

groove - Define un borde ranurado 3D. El efecto depende del valor del color del borde

ridge - Define un borde estriado 3D. EL efecto depende del valor del color del borde

inset - Define - un borde insertado 3D. El efecto depende del valor del color del borde

outset - Define un borde inicial 3D. El efecto depende del valor del color del borde

none - No define bordes

hidden - Define un borde oculto

La propiedad border-style puede tener de uno a cuatro valores (para el borde superior, el borde derecho, el borde inferior y el borde izquierdo).

CSS Border Width

La propiedad border-width especifica el ancho de los cuatro bordes.

Color del borde CSS

La propiedad border-color se utiliza para establecer el color de los cuatro bordes.

El color se puede configurar mediante:

nombre: especifique un nombre de color, como "rojo"

HEX: especifique un valor HEX, como "#ff00ff"

RGB: especifique un valor RGB, como "rgb(0,255,0)"

HSL: especifique un valor HSL,como "hsl(45,100%,50%)"

Bordes individuales

En CSS, también hay propiedades para especificar cada uno de los bordes (superior, derecho, inferior e izquierdo):

Ejemplos de bordes por lados

Border-top-style:dotted;

Border-right-style:dotted;

Border-left-style:dotted;

Bowrder-bottom-style:dotted;

Propiedades abreviadas

La propiedad border es una propiedad abreviada para las siguientes propiedades de borde individuales:

border-width

border-style

border-color

Ejemplo de la abreviacion de propiedades

border: 5px solid red
