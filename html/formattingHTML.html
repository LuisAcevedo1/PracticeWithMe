Formato de texto HTML

HTML contiene varios elementos para definir texto con un significado especial.

Elementos HTML b y strong

El elemento HTML b define texto en negrita, sin importancia adicional.

El elemento HTML strong define el texto con gran importancia. El contenido del interior normalmente se muestra en negrita.

Elementos HTML i y em

El elemento HTML i define una parte del texto en una voz o estado de ánimo alternativo. El contenido interior normalmente se muestra en cursiva.

El elemento HTML em define el texto enfatizado. El contenido interior normalmente se muestra en cursiva.

Elemento HTML pequeños

El elemento HTML small define texto más pequeño:

Este es un ejemplo de texto pequeño

Elemntos HTML de resaltado

El elemento HTML mark define el texto que debe marcarse o resaltarse:

Do not forget to buy milk today.

Elmentos HTML de eliminacion

El del elemento HTML define el texto que se ha eliminado de un documento. Los navegadores suelen marcar una línea en el texto eliminado:

My favorite color is blue red.

Elementos HTML ins

El ins elemento HTML define un texto que se ha insertado en un documento. Los navegadores normalmente subrayarán el texto insertado:

My favorite color is blue red.

Elementos sub en HTML

El elemento HTML sub define el texto del subíndice. El texto del subíndice aparece medio carácter debajo de la línea normal y, a veces, se representa en una fuente más pequeña. El texto con subíndice se puede utilizar para fórmulas químicas, como H2O:

This is subscripted text.

Elementos sup en HTML

El elemento HTML sup define texto en superíndice. El texto en superíndice aparece medio carácter por encima de la línea normal y, a veces, se representa en una fuente más pequeña. Se puede utilizar texto en superíndice para notas a pie de página, como WWW [1]:

This is superscripted text.
