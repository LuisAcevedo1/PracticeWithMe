Atributos en HTML

  • Todos los elementos HTML tiene atributos
  • Los atributos proveen informacion adicional acerca de los elementos
  • Los atributos siempre se especificcan en la etiqueta de inicio
  • Los atributos normalmente vienen en pares nombre/valor como: nombre="valor"

El atibuto href

La etiqueta a define un link. El atributo href especifica la URL donde la pagina va a redirigir con el link

El atributo src

La etiqueta img es usada para mostrar imagenes dentro de la pagina HTML. El atributo src especifica el lugar donde la imagen se encuentra
